o install the base system you just need to get the code from your repo and tell the computer to build it

first you go into your viml folder and type make which reads the files and compiles them into a single executable program

once that finishes you run sudo make install which moves that program into a folder called usr local bin so the computer knows it is a command you can run from anywhere

after that you can just type viml followed by a filename to start editing anything without needing to be in the source folder anymore
